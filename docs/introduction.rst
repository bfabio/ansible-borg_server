Introduction
============

The ``debops.borg_server`` Ansible role can be used to manage a :program:`borg` backup server.

Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.0.0``. To install it, run:

.. code-block:: console

   ansible-galaxy install debops.borg_server
